# Neutrino-mass-from-ultralight-DM
This repository stores codes for analysis of ULDM signatures in neutrino oscillations.

A short description of the files in this repo is given below -

- neutrino_flux.txt: Contains the table of neutrino flux from each reactor at KamLand. Taken from fig. 32 of the paper [arxiv:hep-ph/0107277](https://arxiv.org/abs/hep-ph/0107277).
- survival_probability_new.csv: Neutrino survival probability dataset extracted from the fig. 5 of [arxiv:1303.4667](https://arxiv.org/abs/1303.4667). Also contains the error bars.
- Survival Probability.csv: Neutrino survival probability dataset extracted from the fig. 5 of [arxiv:1303.4667](https://arxiv.org/abs/1303.4667). (OLD DATASET).
- Vacuum Mass Hypothesis.ipynb: Code to fit the data with survival probability in vacuum.
- Dark Mass Hypothesis.ipynb: Code to fit the data with survival probability in the presence of a dark matter.
- Neutrino_Flux.ipynb: To see the neutrino_flux.txt datafile.
- Ultranest Tutorial.ipynb: Tutorial to use UltraNest package.
